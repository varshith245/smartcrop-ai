# SmartCrop AI 🌱

## 📌 Project Description
AI-based system to help farmers with crop recommendations and insights.

## 🚀 Features
- Crop prediction
- Farmer dashboard
- Admin panel

## 🛠 Tech Stack
- Frontend: React
- Backend: Spring Boot
- Database: MySQL

## ▶️ How to Run
1. Clone repo
2. Setup environment variables
3. Run backend & frontend
